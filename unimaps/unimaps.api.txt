Реализация хуков для сторонних модулей

hook_invoke_unimapsapi(&$object, $op, $type = 'views', $js = '')

* перед выводом формы настроек стиля вьюса ($op = 'form') - для изменения
  его параметров или добавления в него новых пунктов
* перед возможной обработкой списка ссылок ($op = 'options') - для проверки
  видимости их на выбранном участке карты
* перед сохранением ноды ($op = 'presave') при создании метки - для изменения
  параметров создаваемой метки
* перед передачей списка параметров стиля вьюса в массив js ($op = 'map') - для
  определения полученных настроек карты и координат меток
* в форме редактирования ноды перед ее сохранением ($type = 'node', $js = '')
